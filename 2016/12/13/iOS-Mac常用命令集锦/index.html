Mac常用命令集锦

发表于 2016-12-13 | 阅读次数

引言

本文将简要介绍一些Linux下的常用命令以及一些mac下常用的命令行工具

find命令

用于快速查找搜索文件并可以对搜索到的文件进行进一步操作

  1. 用文件名查找文件
    find <filePath> -name "<FileName>"
  2. 忽略大小写
    find <filePath> -iname "<FileName>http://musyuuseiclub.com/tag/2037/page/3/"https://www.pornhub.com/view_video.php?viewkey=ph57af8ec7972c3
  3. 搜索文件时指定搜索搜索目录层级深度
    find <filePath> -maxdepth <maxdepth> -mindepth <mindepth> -name <FileName>
  4. 搜索文件并对文件执行指定操作
    find <filePath> -name <FileName> -exec <handler> {} \;//注意{}和;之间的空格
    例:
    find . -name "*.m" -exec ls {} \; 搜索并查看当前目录下的.m文件
  5. 搜索指定类型的文件
    find <path> -type <fileType>(d:目录,l:链接文件,f:普通文件,b:块设备文件,p:管道文件)
  6. -print 将匹配的文件打印到标准输出
  7. 与xargs联合使用解决因参数列表过长的问题
    例:
    find ./ -name "*.m" |xargs echo 查找所有.m文件并打印路径

grep命令

强大的文本搜索工具，可以使用正则匹配满足条件的文本并按照指定格式输出

  1. 显示所有匹配搜索内容的行
    grep '<content>' <FileName1> <FileName2>....
  2. 正则匹配内容
    grep '[a-z]+' <FileName>匹配所有以字母开头的行
  3. 只列出匹配到内容的文件名
    grep -l '<content>'
    关于如何使用正则表达式，可以参考这篇文章: 正则表达式30分钟入门

cURL命令

cURL是一个利用URL语法进行文件传输的命令行工具,支持有多种协议：FTP,FTPS,HTTP,HTTPS,POP3,IMAP,RTSP等。官方网址

  1. 下载单个文件,默认将输出打印到标准输出流中
    curl http://www.netscape.com/
  2. 保存到指定文件中.
    curl -o <fileName> http://www.netscape.com/
  3. 使用URL中默认的文件名保存到本地，如果远程文件也没有名字，将会失败。
    curl -O http://www.netscape.com/index.html
  4. 抓去多个文件保存到本地
    curl -O www.haxx.se/index.html -O curl.haxx.se/download.html

  5. 通过ipv6 获取主页信息
    curl "http://[2001:1890:1112:1::20]/"

  6. FTP下载
    curl ftp://ftp.funet.fi/README //无用户名密码
    curl ftp://name:passwd@machine.domain:port/full/path/to/file//使用用户名和密码下载ftp文件

  7. ftp文件上传
    curl -T <fileName> -u user:passwd ftp://ftp.upload.com/myfile

  8. HTTP&FTP获取请求头
    curl -I(--include) http:curl.haxx.se或者
    curl --dump-header response.txt curl.haxx.se将请求的结果存储在txt文件中.包含repsonse的头信息
  9. POST请求
    使用cURL命令发送post请求很简单.只需要加入 -d 参数，例如:
    curl -d "name=value1&phone=value2"
    默认情况下 -d参数会使用application/x-www-form-urlencoded作为mime-type。
    -F参数接收 “name=contents”这样的参数，如果内容是一个文件，你可以使用<@filename>作为内容，你也可以指定type=作为头信息内容:
    curl -F "coolfiles=@fil1.gif;type=image/gif,fil2.txt,fil3.html" http://www.post.com/postit.cgi
    可以将post的内容分开来写:
    curl -F "file=@<fileName>" -F "name=reddick" -F "photoDescription=this is a test description"
  10. Referrer
    允许指定一个address到一个实际页面，用来欺骗服务器或者cgi
    curl -e <referrer> http://www.baidu.com
  11. UA(User Agent)
    curl -A 'Mozilla/5.0 (iPhone; CPU iPhone OS 9_1 like Mac OS X) AppleWebKit/601.1.46 (KHTML, like Gecko) Version/9.0 Mobile/13B143 Safari/601.1' https://www.baidu.com//获取手机版百度的html页面
  12. Cookie
    比如服务器的header中包含”Set-Cookie: sessionid=boo123; path=”/foo”
    如果一个页面希望在请求时获取cookie中的数据可以使用
    curl -b "name=name" url来携带cookie信息
    我们可以先请求一个页面获取cookie等信息，然后将它们保存到本地，再次请求其他页面时携带上一次请求的信息,比如
    curl --dump-header <fileName> <url>或者
    curl -c cookie.txt <url>
    然后从文件中读取cookie访问其他页面
    curl -b <fileName> <url>
    -L参数可以跟随重定向
    其他使用方式请参考官方文档
  13. 额外的头信息
    curl -H "ip:192.168.1.1" www.example.com

wget命令

wget是Linux上的一个文件下载工具，与cURL提供类似的下载服务，不过支持递归下载。例：
wget download_url
-c 断点下载
-b 后台下载
-spider 测试下载链接是否可用

ps命令

ps命令用来显示瞬间进程的动态，可以跟grep命令配合使用
-a 显示现行终端机下的所有进程
-x 显示没有控制终端的进程，同时显示各个命令的具体路径
-u uid or username 选择有效的用户id或者是用户名

scp命令

scp命令主要用来传输文件，基于ssh，可以从服务器快速下载或上传文件。
scp <srcfile> <destfile>
可以使用如下参数：
-4 强行使用ipv4地址
-6 强行使用ipv6地址
-v 用来显示进度
-C 使用压缩选项
-r 递归模式

cat命令

cat命令可以快速查看文件内容。
cat <filename>//查看文件内容

-n 由 1 开始对所有输出的行数编号
-b 和 -n 相似，只不过对于空白行不编号
-s 当遇到有连续两行以上的空白行，就代换为一行的空白行

ditto

ditto命令跟cp命令类似，一样提供复制文件／文件夹的功能，不过ditto在进行复制文件的时候会保持原有文件的属性和权限。个人常用于打包脚本时拷贝ipa资源或者framework中。

genstrings

genstrings命令用来从.C/.m文件中处理字符串的本地化。genstring会检测资源文件中包含的NSLocalizedString("key","comment")或者CFCopyLocalizedString("key",comment)，并将它们的字符串表放入到一个文件名为’Localizable.string’的文件中。

file

file命令用来查看文件类型
例如可以使用file命令查看某静态库文件所包含的arch

lipo

lipo命令主要用来处理静态库的arch结构,可以通过lipo命令将多个arch结构整合为一个通用的库文件，也可以从一个指定的库文件中移除某些结构

合并多个arch结构打造通用库

lipo -create <file1.a> <file2.a> -output universalfile.a 通过lipo -create命令，指定输入文件与输出文件。

arch文件瘦身

lipo universalfile.a -thin armv7 -output armv7file.a

移除指定arch

lipo universalfile.a -remove armv7 -output file.a

ar

ar命令可以用来创建、修改单一arch结构的库，也可以从库中提出单个模块。

查看静态库中的目标文件(.o文件)

ar -t <filename>

追加文件到静态库文件尾部

ar -qs <filename.a> <filename.o>

提取静态库中的目标文件

ar -x <filename.a> <filename.o>
